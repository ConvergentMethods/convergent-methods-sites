Boyce

Privacy-first SQL compiler and open semantic protocol, exposed as an MCP server. Gives AI agents the structured database intelligence they need to generate correct, safe SQL — deterministically.

Named for Raymond F. Boyce, co-inventor of SQL (1974) and co-author of Boyce-Codd Normal Form.
